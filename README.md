# OpenClaw Config

這是 OpenClaw AI 助理的設定檔案倉庫。

## 檔案說明

- `openclaw.json` - OpenClaw 主要設定檔
- `.env.sample` - 環境變數範本（複製為 `.env` 並填入真實值）
- `.gitignore` - Git 忽略規則

## 使用方法

1. 複製 `.env.sample` 為 `.env`
2. 填入你的 API keys 和 token
3. 將設定檔案放到 `~/.openclaw/` 目錄

## 安全提醒

⚠️ **絕對不要將 `.env` 檔案 commit 到 git！**
